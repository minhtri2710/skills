# Critique Guide

Use this when the user asks for a review, rating, or design critique.

## Score Five Dimensions

1. Philosophical coherence
2. Visual hierarchy
3. Execution quality
4. Functional clarity
5. Originality

Score each from 0 to 10.

## Output Format

Use:

- overall verdict
- score table
- keep
- fix
- quick wins

## Severity Labels

- `fatal`: blocks the design from succeeding
- `important`: should be fixed before delivery
- `polish`: worth improving if time allows

## What Good Critique Sounds Like

- specific
- visual
- actionable
- detached from the designer's ego

## What To Flag

- no clear focal point
- generic or derivative mood
- mismatch between concept and treatment
- brand inconsistency
- beautiful but unusable interaction
- polished surface hiding weak structure
