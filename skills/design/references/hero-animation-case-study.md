# Hero Animation Case Study

Use this pattern when the scene must express both breadth and focus.

## Preconditions

- many related assets or states
- a need to show scale without flattening everything
- a hero section or launch sequence, not a utility demo

## Pattern

1. broad field appears
2. attention ripples across the field
3. focus narrows onto a few selected elements
4. those elements become the actual narrative anchors

## Benefits

- communicates “many” and “chosen” in one sequence
- keeps density readable
- gives the reveal a strong payoff

## Use Carefully

If the scene only has a few assets, this pattern is unnecessary.
