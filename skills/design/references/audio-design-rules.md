# Audio Design Rules

Use audio only when it improves the piece.

## Two Layers

- BGM for atmosphere
- SFX for structure and emphasis

## Density Guide

- launch hero: moderate cue density
- tool demo: sparse cues
- calm editorial motion: maybe no SFX at all

## Good Cue Targets

- major reveal
- state confirmation
- transition hit
- branded closing gesture

## Avoid

- a sound for every tiny movement
- loud repetitive clicks
- cues that fight the BGM rhythm
- comedic sounds on serious work

## Build A Cue List

Write:

- timestamp
- event
- cue type
- intensity

Then map the cue list to sourced files or the project's own library, using `sfx-library.md` as the taxonomy. The bundled `assets/` audio files are BGM only.
