# SFX Library

This file is a cue taxonomy, not a bundled asset library. The skill ships BGM tracks in `assets/`, but it does not ship SFX files.

## Categories

- `feedback`: confirmations, success, error, notification
- `impact`: brand reveal, stamp, landing thud
- `keyboard`: typing and terminal rhythm
- `magic`: transform, sparkle, AI-process feel
- `progress`: loading, ticking, completion
- `terminal`: command execution and output appearance
- `transition`: whoosh, slide, dissolve
- `ui`: click, toggle, hover, focus

## Selection Rule

Choose by narrative role, not by novelty.

Examples:

- command line demo: `keyboard` plus `terminal`
- polished UI interaction: mostly `ui`
- launch hero: `impact` plus a few `transition` cues
- speculative concept motion: maybe `magic` plus sparse `impact`

## Sourcing Rule

Choose or source real SFX files that match the category and tone. Record the selected file paths in your cue list or delivery notes so the next pass can reproduce the mix.
