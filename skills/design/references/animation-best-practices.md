# Animation Best Practices

This file is for motion design principles and language.

## Narrative Shape

A strong short-form motion piece usually follows:

1. premise
2. build
3. reveal
4. payoff
5. brand or conclusion hold

## Motion Language

Use a limited vocabulary and repeat it:

- rise and settle
- sweep and reveal
- soft counter-motion
- measured scaling
- focus shift

## Timing

- open quickly
- give the main reveal room to breathe
- compress repetitive transitions
- leave a clean ending hold for export

## Easing

Good defaults:

- strong out-ease for primary entrances
- gentle in-out for panels and state changes
- overshoot sparingly for playful controls

## Composition

- design each beat like a poster
- keep one active focal area
- dim or simplify what is not active

## Sound Pairing

If the output includes sound:

- map cues to narrative beats
- keep density proportional to the style
- do not turn every motion into a sound effect
